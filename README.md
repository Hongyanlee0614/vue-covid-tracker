# vue_covid_tracker

## Project setup

```
vue create vue_covid_tracker
cd vue_covid_tracker
vue add tailwind
npm install moment
```

### Compiles and hot-reloads for development

```
npm run serve
```

Open in localhost:8080

### Compiles and minifies for production

```
npm run build
```
